The KerbalDeaths parameter _fails_ if more Kerbals than the countMax die.

<pre>
PARAMETER
{
    name = KerbalDeaths
    type = KerbalDeaths

    // Maximum Number of Kerbals that can die before this contract is considered
    // failed.
    //
    // Type:      <a href="Numeric-Type">int</a>
    // Required:  No (defaulted)
    // Default:   1
    //
    countMax = 1
}
</pre>
