##### VesselIsType
Checks the VesselType of the given vessel

    PARAMETER
    {
        name = VesselIsType
        type = VesselIsType

        // Type of vessel to check for.  Valid values are a subset from the
        // VesselType enum:
        //   Base
        //   EVA
        //   Lander
        //   Probe
        //   Rover
        //   Ship
        //   Station
        //   SpaceObject
        vesselType = EVA
    }
